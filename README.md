### NervFox

A Neon Genesis Evangelion–inspired Firefox theme.

Retro Win95/98 look with NERV styling for Firefox. Includes Firefox Quantum logo* and alternate color schemes in `colors.txt`.

Forked from Firefox-98.

## Installation

1. Clone the theme:
   ```bash
   git clone https://github.com/viztini/NervFox.git
   cd NervFox
``

2. Make the installer executable:

   ```bash
   chmod +x install.sh
   ```

3. Run the installer:

   ```bash
   ./install.sh
   ```

4. In Firefox, open `about:config` and set:

   ```
   toolkit.legacyUserProfileCustomizations.stylesheets = true
   ```

5. Restart Firefox
